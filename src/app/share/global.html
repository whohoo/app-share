Javascript

这里定义了4个全局变量，并使用当前的URL进行赋值，开发中可根据实际需要进行修改。

window.__shareDef = {base_url:window.location.href.match(/http[\w:\.\/-]+\w\//)[0]}; __shareDef.title = ""; // 分享标题 __shareDef.desc = ""; // 分享描述 __shareDef.img = __shareDef.base_url + "img/share.jpg"; // 分享图标 __shareDef.url = "" || __shareDef.base_url; // 分享链接
