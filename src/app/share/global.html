定义全局变量

定义了4个全局变量，并使用当前的URL进行赋值，开发中可根据实际需要进行修改。

/***********定义__shareDef全局变量**************/ window.__shareDef = {base_url:window.location.href.match(/http[\w:\.\/-]+\w\//)[0]}; __shareDef.title = ""; // 自定义分享标题 __shareDef.desc = ""; // 自定义分享描述 __shareDef.img = __shareDef.base_url + "img/share.jpg"; // 自定义分享图片 __shareDef.url = "" || __shareDef.base_url; // 自定义分享链接 || 当前网页URL
